ipmi-reading-boolean-match
Boolean match test
boolean_match
481
94
30
Alpha
30
0

Documentation:
The program looks at fields marked as boolean. It looks at the text in this

field. If it matches the text in the IPMI then the value is set to TRUE.

if not the value is set to FALSE.

Triggers are raised when the value of the field is false.

So the description for a redundant power supply sensor should be

"Power supply OK". The reading true value should be "ok". Whenever

the reading returns something other than OK. Then the value is set

to FALSE and a trigger alarm raised.
